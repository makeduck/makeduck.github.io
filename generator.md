Generator
============================

개요
-----------------
코드는 만드는 코드를 만들어서 개발을 안해보자.

mybatis generator
-------------------
- db schema 통해서 model 생성
- crud mapper 생성
- Criteria model 생성

방법
-----------------
mybatis generator project 를 수정한다.
service, conroller 영역까지 code generate 한다.
서버스를 통째로 찍어서 개발할 수 있게 한다.
